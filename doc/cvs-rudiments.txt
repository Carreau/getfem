Principales commandes de cvs utilisées pour la gestion du projet
----------------------------------------------------------------


cvs add nom_fichier :	ajout d'un fichier dans cvs. On doit se positionner sur
			le repertoire ou est le fichier
cvs remove nom_fichier : suppression d'un fichier. On doit se positionner sur
			le repertoire ou est le fichier
cvs commit -m "commentaire" : validation de changements effectués sur	
			les fichiers
cvs import -m "commentaire" nom_repertoire getfem start : On doit se
			positionner  sur le repertoire ou se trouvent les
			fichiers à intégrer.
