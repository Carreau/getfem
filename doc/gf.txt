

Structures essentielles de Getfem (coeur de la bibliothèque):


- Description des elements de référence et transformation géométrique

- Description d'un maillage

- Description de parties de maillages

- Description des méthodes éléments finis sur un élément
  (en général de référence)

- Description d'une méthode élément fini sur un maillage.

- Procédures d'assemblages.
