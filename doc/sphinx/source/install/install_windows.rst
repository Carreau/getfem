How to install from sources on Windows
======================================



See the `download and install <../download.html>`_ page for general requirements and the download of the last stable version of |gf|.

On windows, there is several ways to build |gf| library. Either with a commercial C++ compiler or with Mingw or Cygwin. For MSCV, some projects files are available on the git version inside the ``mscv`` directory.
